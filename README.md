# Калькулятор геометрических фигур
## 
Консольное приложение для расчета параметров геометрических
фигур с сохранением истории расчетов в базу данных SQLite 
и экспортом результатов в документы Word.
### Возможности:
- Расчет площади, радиусов описанной и вписанной окружностей для:
  - Прямоугольников (включая квадраты)
  - Треугольников (по трем сторонам)
  - Трапеций
- Сохранение истории расчетов в базу данных SQLite
- Экспорт отчетов в формат Word (.docx)
- Просмотр истории расчетов с фильтрацией и поиском

## Инструкция по запуску:
-Скачайте или склонируйте файлы проекта
-Откройте терминал в папке с проектом
-Запустите главный файл приложения:
```bash
python interface.py
```
Для запуска тестов pytest
```bash
python pytests.py
```
## Запуск через докер:
1. Соберите образ

````bash
docker build -t geometry-calculator .
````
2. Запустите Докер

```bash
# Простой запуск
docker run -it --rm geometry-calculator

# С сохранением базы данных на хосте
docker run -it --rm -v C:\geometry_data:/app geometry-calculator

# С полным доступом к файлам
docker run -it --rm -v ${PWD}:/app geometry-calculator
```
